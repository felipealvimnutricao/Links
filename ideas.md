# Design Brainstorm - Felipe Alvim Links Hub

## Contexto
Site de links para nutricionista Felipe Alvim. Paleta de cores: **Preto + Dourado**. Objetivo: Moderno, sofisticado e fácil de gerenciar banners/links.

---

## Opção 1: Luxo Minimalista (Probabilidade: 0.08)

**Design Movement:** Luxury Minimalism / High-End Simplicity

**Core Principles:**
- Espaço negativo generoso como elemento de design
- Tipografia elegante e hierarquizada
- Contraste dramático entre preto profundo e dourado brilhante
- Transições suaves e refinadas

**Color Philosophy:**
- **Fundo:** Preto profundo (#0a0a0a) com gradiente sutil para cinza escuro
- **Destaque:** Dourado metálico (#d4af37) apenas em elementos críticos
- **Texto:** Branco puro (#ffffff) com cinza claro (#e0e0e0) para secundário
- **Intenção:** Sofisticação, exclusividade, confiança profissional

**Layout Paradigm:**
- Hero section com foto do Felipe em diagonal/assimétrica
- Banners em grid 1-coluna com espaçamento vertical generoso
- Cada banner ocupa 80-90% da viewport com padding lateral
- Seção de redes sociais no rodapé com ícones minimalistas

**Signature Elements:**
- Linha dourada horizontal como divisor entre seções
- Ícones minimalistas em dourado para cada banner
- Efeito hover com brilho dourado sutil (glow)

**Interaction Philosophy:**
- Hover: Brilho dourado suave, elevação do card
- Click: Transição rápida (200ms) com feedback visual
- Scroll: Parallax leve na imagem de fundo

**Animation:**
- Entrada: Fade-in com slide up (300ms, ease-out)
- Hover: Scale 1.02 + glow dourado (200ms)
- Transição de página: Fade suave (150ms)

**Typography System:**
- Display: Playfair Display (bold) para títulos
- Body: Lato (regular/light) para textos
- Hierarchy: 48px (h1) → 24px (h2) → 16px (body)

## Opção 2: Dark Modern Gradient (Probabilidade: 0.07)

**Design Movement:** Contemporary Dark Mode / Glassmorphism

**Core Principles:**
- Gradientes sutis como base visual
- Efeito glassmorphism em cards (blur + transparência)
- Animações fluidas e responsivas
- Foco em modernidade e movimento

**Color Philosophy:**
- **Fundo:** Gradiente preto → cinza escuro (#0a0a0a → #1a1a1a)
- **Cards:** Glassmorphism com backdrop-filter (blur 10px)
- **Destaque:** Dourado com gradiente para laranja (#d4af37 → #ff9500)
- **Intenção:** Contemporâneo, dinâmico, tech-forward

**Layout Paradigm:**
- Hero com imagem em corner (canto superior direito)
- Banners em cards com borda sutil dourada
- Efeito de profundidade com múltiplas camadas
- Rodapé com ícones animados

**Signature Elements:**
- Borda dourada animada em hover
- Ícones com animação de rotação leve
- Fundo com padrão geométrico sutil (grid/dots)

**Interaction Philosophy:**
- Hover: Borda brilha, fundo se ilumina
- Click: Ripple effect em dourado
- Scroll: Cards entram com stagger animation

**Animation:**
- Entrada: Slide from bottom + fade (400ms, cubic-bezier)
- Hover: Border glow + background brighten (300ms)
- Ícones: Rotate 360° on hover (600ms)

**Typography System:**
- Display: Montserrat Bold para títulos
- Body: Open Sans para textos
- Hierarchy: 56px (h1) → 28px (h2) → 16px (body)

---

## Opção 3: Organic Elegance (Probabilidade: 0.06)

**Design Movement:** Organic Design / Biophilic Luxury

**Core Principles:**
- Formas orgânicas e curvas naturais
- Integração de elementos naturais (folhas, texturas)
- Movimento fluido e natural
- Sofisticação através da natureza

**Color Philosophy:**
- **Fundo:** Preto com textura orgânica (ruído natural)
- **Destaque:** Dourado com variações (ouro velho, bronze)
- **Acentos:** Verde menta (#a8d5ba) para elementos secundários
- **Intenção:** Bem-estar, natureza, saúde holística

**Layout Paradigm:**
- Hero com forma orgânica (wave/blob) contendo foto
- Banners com bordas arredondadas e suaves
- Divisores em forma de onda (SVG)
- Fundo com padrão de folhas sutis

**Signature Elements:**
- Formas blob animadas como background
- Ícones com linhas orgânicas
- Transições em forma de onda

**Interaction Philosophy:**
- Hover: Forma expande suavemente
- Click: Ripple orgânico
- Scroll: Parallax com movimento natural

**Animation:**
- Entrada: Morph from blob (500ms, ease-in-out)
- Hover: Expand + color shift (300ms)
- Background: Subtle float animation (4s loop)

**Typography System:**
- Display: Cormorant Garamond para títulos
- Body: Poppins para textos
- Hierarchy: 52px (h1) → 26px (h2) → 16px (body)

---

## Decisão Final

**Escolhido: Opção 1 - Luxo Minimalista**

Razão: Melhor alinhamento com a marca de um nutricionista profissional. Transmite confiança, sofisticação e exclusividade. O espaço negativo generoso mantém o foco nos banners/CTAs, facilitando a conversão. Mais fácil de manter e gerenciar.
